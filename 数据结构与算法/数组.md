# C#数据结构之动态数组
首先先来看下C#中默认包含的数组有那些:
* 静态数组
	* 特点:静态数组一旦初始化后,容量就是有初始化来固定的,后续容量的大小是无法改变的,如果Count>Length,那么静态数组就存在越界,只能声明一个新的数组来存放新的数据
	```csharp
	int []
	float [] 
	double []
	char [] 
	string [] 
	```
* 动态数组
	* 特点:动态数组可以根据数组中实际元素的数量来决定并动态的调整数组容量的大小,相比静态数组,性能更优
	``` csharp
	//以下两种数组是由C#系统自带默认提供的
	ArrayList(Object) 
	List<T>(Generic)
	```

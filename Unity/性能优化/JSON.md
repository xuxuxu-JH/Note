# 数据持久化JSON
## Json的语法格式,以及如何使用json来表示数据
```json
//Json中的注释形式和C#中的一致,支持单行和多行注释
//Json单行注释
/* 多行注释
*/
{
    //json支持的数据类型:
    //值类型:整数或浮点
    //字符串
    //bool
    //数组 对象 null
    //json用键值对的形式来表达数据
    //>json中的符号基本含义:
    //大括号{} --- 表示对象
    //中括号[] --- 表示数组
    //冒号:代表的是键值对的对应关系
    //逗号就是分割成员变量的间隔符,json中最后一个字段的逗号需要省略
    //>json当中的"键"一定用双引号
    "name": "黄中旭",
    "age": "21",
    "sex": true,
    //表示浮点类型
    "testFloat": 1.3,
    //>中括号代表数组
    "array": [
        1,
        2,
        3,
        4,
        5,
        6
    ],
    //Students数组存储的是Person的类的类型的数组
    //假如Students数组中存储了两个Person类的对象数据,里面使用两个中括号来表示就好了
    "Students": [
        {
            "name": "小红",
            "age": "10",
            "sex": false
        },
        {
            "name": "小明",
            "age": "8",
            "sex": true
        }
    ],
    //>json表示类
    //>表示一个Home类对象,类中的字段名和成员类型都要一一对应
    "home": {
        "address": "广州",
        "street": "大沙地"
    },
    //表示Person类对象son,和类中的成员字段一一对应,和上文中的Student表达的方式一致
    "son": {
        "name": "小强",
        "age": "12",
        "sex": true
    },
    //表示一个空对象 null赋值
    "son2": null,
    //json表示一个字典,将字典理解为对象的概念即可
    //>由于json的特殊规则,键只能是双引号,也就是string
    "dic": {
        //键值表示字典中的成员
        "1": "123",
        "2": "456"
    }
}
```

## C#读取Json文件
### JsonUtility
JsonUtility是Unity自带的解析JSON的公共类,它主要提供序列化和反序列化两个功能
1. 将内存中的对象序列化为Json格式的字符串
2. 将Json字符串反序列化为类对象
